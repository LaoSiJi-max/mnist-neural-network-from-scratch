# MNIST Neural Network from Scratch

[中文](README.md) | [日本語](README_ja.md)

NumPyを使用して手動で実装した、MNIST手書き数字分類用ニューラルネットワークです。

## プロジェクト紹介

大学院在学中の「人工知能のための数学」という授業で取り組んだ自由課題です。

この授業では、先生が執筆された教材『最短コースでわかる ディープラーニングの数学』を使って学習しました。ニューラルネットワークの数学的な仕組みとプログラムでの実装がとても分かりやすく結び付けられていたため、自分なりの理解に基づいて、ニューラルネットワークを実際に一から書いてみたいと思いました。

最終的に、このNotebookを授業の期末自由課題として提出し、先生から高い評価をいただきました。

「追加課題、素晴らしいレポートでありがとうございます！ルールで最大20点なのですが、このレポートだけで100点をあげたい気持ちです。」

先生のご指導に感謝しています。教材も本当に素晴らしい内容でした。

このNotebookは教材の内容を参考にしながら、必要に応じて自分でも資料を調べ、順伝播、逆伝播、勾配降下法について理解した内容をもとに、少しずつ実装したものです。

データセットの読み込みと前処理には既存のライブラリを使用していますが、順伝播、逆伝播、損失計算、パラメータ更新など、ニューラルネットワークの主要な学習処理はNumPyを使って自分で実装しています。

## 実装内容

* MNIST手書き数字データセットの読み込みと前処理
* 28×28画像の784次元入力への変換
* 入力データの正規化
* 数字ラベルのOne-Hot表現への変換
* ReLU活性化関数の手動実装
* Softmax関数の手動実装
* 交差エントロピー損失関数の手動実装
* 順伝播処理の手動実装
* 逆伝播処理の手動実装
* ミニバッチ勾配降下法によるパラメータ更新
* 学習中の損失値と正解率の記録・表示
* テストデータを使用した最終評価

## ネットワーク構成

2つの隠れ層を持つ全結合ニューラルネットワークを使用しています。

```text
入力層：784
   │
   ▼
隠れ層：128 + ReLU
   │
   ▼
隠れ層：128 + ReLU
   │
   ▼
出力層：10 + Softmax
```

出力層の10個のノードは、数字の0から9に対応しています。

主な学習パラメータは以下のとおりです。

| パラメータ      | 設定                          |
| ---------- | --------------------------- |
| 学習回数       | 100 Epochs                  |
| 学習率        | 0.01                        |
| Batch Size | 512                         |
| 隠れ層のノード数   | 128、128                     |
| 最適化方法      | Mini-batch Gradient Descent |

## 逆伝播の実装

この課題の中で、自分なりの工夫を最も加えた部分は逆伝播です。

各層に対して固定的な逆伝播コードを個別に書くのではなく、出力層から入力側へ向かってループ処理を行い、各層の勾配計算と重み更新を順番に実行する形にしました。

```python
for j in range(len(w) - 1, -1, -1):
    w_old = w[j][1:].T
    w[j] = w[j] - alpha * (
        inputs[j].T @ y_diff / inputs[j].shape[0]
    )

    if j > 0:
        y_diff = step(linear_result[j - 1]) * (
            y_diff @ w_old
        )
```

このようにした目的は、逆伝播処理を固定されたネットワークの層数に依存させないためです。

同じ形式で重み行列を追加すれば、ネットワークの層を増やした場合でも、逆伝播の主要部分を各層ごとに書き直す必要がありません。

現在の視点で見ると、このコードにはまだ改善できる部分があると思いますが、当時の自分にとっては、理解した内容を自分の手で実装できたことに最も達成感を感じた部分です。
