# MNIST Neural Network from Scratch

[中文](README.md) | [日本語](README_ja.md)

使用 NumPy 手动实现的 MNIST 手写数字分类神经网络。

## 项目介绍

这是读研期间“人工知能のための数学”课程中的一份自由课题。

在这门课中，我学习了老师编写的教材《最短コースでわかる ディープラーニングの数学》。书中把神经网络的数学原理和程序实现联系得很清楚，因此我想试着按照自己的理解，把一个神经网络真正写出来。最后，我把它作为这门课的期末自由课题提交，老师给出了很高的评价：“虽然评分规则规定最高只能给20分，但我真想光凭这份报告就给100分。”在此也感谢老师的教导，教材确实写得非常好。

这个 Notebook 以教材中的内容做参考，一边阅读教材，一边查找资料，再根据自己对顺向传播、反向传播和梯度下降的理解逐步完成实现。

数据集的读取和预处理使用了现有库，但神经网络的主要学习过程，包括顺向传播、反向传播、损失计算和参数更新，都是使用 NumPy 自己实现的。

## 实现内容

* 读取并预处理 MNIST 手写数字数据集
* 将 28×28 的图像转换为 784 维输入
* 对输入数据进行归一化
* 将数字标签转换为 One-Hot 编码
* 手动实现 ReLU 激活函数
* 手动实现 Softmax 函数
* 手动实现交叉熵损失函数
* 手动实现顺向传播
* 手动实现反向传播
* 使用小批量梯度下降更新参数
* 记录并显示训练过程中的损失和准确率
* 使用测试数据进行最终评价

## 网络结构

本项目使用了一个包含两个隐藏层的全连接神经网络：

```text
输入层：784
   │
   ▼
隐藏层：128 + ReLU
   │
   ▼
隐藏层：128 + ReLU
   │
   ▼
输出层：10 + Softmax
```

输出层的10个节点分别对应数字0到9。

主要训练参数如下：

| 参数         | 设置                          |
| ---------- | --------------------------- |
| 训练轮数       | 100 Epochs                  |
| 学习率        | 0.01                        |
| Batch Size | 512                         |
| 隐藏层节点数     | 128、128                     |
| 优化方式       | Mini-batch Gradient Descent |

## 反向传播的实现

这份作业中，我自己发挥最多的部分是反向传播。

在实现时，我没有为每一层分别写固定的反向传播代码，而是使用循环从输出层开始，依次向前计算各层的梯度并更新权重：

```python
for j in range(len(w) - 1, -1, -1):
    w_old = w[j][1:].T
    w[j] = w[j] - alpha * (
        inputs[j].T @ y_diff / inputs[j].shape[0]
    )

    if j > 0:
        y_diff = step(linear_result[j - 1]) * (
            y_diff @ w_old
        )
```

这样设计的目的，是让反向传播过程不依赖固定的网络层数。

只要按照相同方式增加权重矩阵，反向传播的主体逻辑就不需要为每一层重新编写。对现在的我来说，这段代码可能仍然有很多可以改进的地方，但这是当时自己理解和实现得最有成就感的一部分。
